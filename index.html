I Love Javascript Coding Africa

JS4Afrika is an educational non-profit Group that promotes web programming and JavaScript in Africa for developers and students at all experience levels.

We are One Family

JS4Afrika is a multinational community run be javascript enthausiasts in different parts of Africa. Periodically we get people together in our respective communities and help each other hack cool web apps. Dates and locations are announced on the js4africa mailing list as well as each country's individual mailing list.
